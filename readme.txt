Automatically generates a Google News Sitemap using the Google News Sitemap specification.

== Description ==

This plugin generates an XML sitemap to Google News (https://support.google.com/news/publisher/answer/74288?hl=pt-BR) according to the settings made on the panel 

Some features:

* Set default values on a dedicated settings page
* Set genre, access, language on per article basis
* Exclude articles from the sitemap
* Ping Webmaster Tools when sitemap is updated

If you find a bug or you have a feature request, feel free to mail the developer.

You can use the built in installer and upgrade, or you can install the plugin
manually.

== Installation ==

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
2. Disable any other News Sitemap generator plugin you may have installed!
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Visit the Google News Sitemap settings (*Settings -> Google News Sitemap*)
5. Configure the options.
6. Login to your Webmaster Tools account and submit the newly created news sitemap
